# RescueMind Prototype

> **Implementation repository for the RescueMind research project.**

RescueMind-Prototype contains the software implementation, simulation environments, algorithms, and experimental prototypes that support the RescueMind research vision.

## Related Research Repository

The scientific vision, system architecture, research objectives, and conceptual framework are maintained in the main repository:

**https://github.com/panagiotagrosdouli/RescueMind**

This repository focuses on implementation rather than documentation.

## Objectives

- Develop modular AI components for disaster response.
- Build multi-agent robotic simulations.
- Implement sensor fusion pipelines.
- Develop the Rescue Priority Index (RPI).
- Prototype the Living Disaster Twin.
- Evaluate algorithms under realistic disaster scenarios.

## Planned Technology Stack

- Python
- ROS 2
- OpenCV
- PyTorch
- NumPy
- Open3D
- Gazebo / Isaac Sim
- Docker
